高等代数

AX=c 是一个未知数2个，方程3个的方程组 ，不一定有解 ，这个问题相当于； $a_1$ , $a_2$ 作为基构成了一个三维空间的一个平面 ，如果 c 在这个平面上 ，那么有解 ，如果不在 ，则无解 。为了寻找最贴合的解 ，首先 ，我们定义什么来刻画贴合程度 ， 面上的点到 c 的距离 ，因为是三维的 ，就是三个坐标差 。 左边式子就是说面上的点和 c 连成的直线和面垂直 ，此时距离最小 ，转化一下得到右边式子 ，发现右边式子有解 ，而且根据前面的推断 ，它是最吻合的 。 (1) 的右边就是求最吻合的解，而左边就是垂直的条件 ，该题主要通过勾股定理证大小关系 ，具体就是证明垂直关系 。 (2) 是齐次方程组同解 ，证法就是这边的解是那边的解 ，一一对应 。两个矩阵的秩相等 ，需要构造两个不等式 。从 (1) 题的左边方程的增广矩阵推导 ，一个不等式通过矩阵合并后的秩不会变小得到 ，就是增广矩阵的意思 。同时 (2) 左边证明了两个矩阵式同秩的 。另一个不等式从增广矩阵可以拆分成矩阵相乘得到 ，而矩阵相乘秩不变大 。 从垂直条件得到比例 ，再从比例推导到勾股定理 。

cloudflare saas 加速站点

需要 2 个域名 ，一个托管到 cloudflare ，一个不能托管到 cloudflare 。给访问者显示的是主域名 ，用来回源的是工具域名 。接下来我展示 origin.lxz.cloudns.be 是工具域名 ，blog.0031400.xyz 是主域名 。 origin.lxz.cloudns.be 解析到网站的 ip 地址 ，或者域名 ，开启 cdn 代理 。 ssl/tls 的自定义主机名选项栏 ，添加回退源 origin.lxz.cloudns.be 添加自定义主机名 填写主域名 ，我的 blog.0031400.xyz 已经在使用 ，现在用 test.0031400.xyz 代替演示 。 添加两条 txt 记录在主域名的 dns 系统 等待结果完成 主域名指向一个域名 ，这个域名托管平台无要求 ，这样做是为了实现国内外分流加速 。 我这里主域名指向 cdn.0031400.xyz ，cdn.0031400.xyz 分两条 ，默认指向一个 cloudflare 的优选 ip ，境外线路指向 1.0.0.5 成果看 ，是不错的 ，解析失败的那些地区是因为我昨晚换了一个 dns 解析 ，目前存在缓存 。
