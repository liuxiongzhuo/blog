hugo 创建博客网站

今天使用 hugo 的 paperMod 主题做博客放在 github pages 上面。下面我来介绍一下我的配置过程。 安装 hugo https://github.com/gohugoio/hugo 我电脑是 win11 系统 ，我在 github hugo 官网下载 hugo 的 exe 可执行文件 。 把 hugo 的可执行文件添加到系统的环境变量 。 hugo new site 网站名字创建一个 hugo 网站项目 。 git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1 进入项目目录 ，安装 paperMod 主题 。 因为我是默认使用 yaml 配置模式 ，所以我使用 yaml 语法 ，在 hugo.yaml 添加 theme: ["PaperMod"] 我贴一贴我目前的配置文件 baseUrl: 'https://blog.0031400.xyz' # 网站地址 ，也可以省略 ，此时使用相对路径 languageCode: 'zh-CN' # 网站语言 title: '0031400 的博客' # 网站题目 theme: ["PaperMod"] # 网站主题 params: # 下面是参数 title: '0031400 的博客' # 显示在 html head 的 title description: '分享生活 学习' # 显示在 html head 的 description keywords: ['大学','学习','生活','博客'] # 显示在 html head 的 关键词 author: '0031400' # 显示在 html head 的 作者 DateFormat: '2006-01-02' # 网站内日期显示格式 ShowCodeCopyButtons: true # 代码块显示复制按钮 homeInfoParams: # paperMod 的配置 ，分三种 ，我选择 homeInfo Title: '0031400 的博客' # homeInfo 的 title Content: '分享生活 学习' # homeInfo 的 显示内容 socialIcons: # homeInfo 显示的社交图标 - name: github url: 'https://github.com/liuxiongzhuo' - name: email url: 'mailto://0031400@0031400.xyz' assets: # 在 github page 发布的必要设置 ，我也不懂 ，如果没有这个 ，是显示不了 css 的 。 disableFingerprinting: true publishDir: docs # 编译输出目录 ，我是在 github page ，为了方便 ，我设置为 docs menu: # 菜单栏 main: - identifier: archives name: 归档 url: /archives/ weight: 10 - identifier: search name: 搜索 url: /search/ weight: 40 outputs: # 设置搜索页要求的 home: - HTML - RSS - JSON 在 content 的 posts 里面的文章是显示在网页首页的文章 。 ...
